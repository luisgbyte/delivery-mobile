## :game_die: How to run the project?
```
# Clone this repository
$ git clone https://github.com/luisgbyte/delivery-mobile

# Access the repository on your terminal
$ cd delivery-mobile

# Install dependencies
$ yarn

# Run the project
$ yarn start
```
